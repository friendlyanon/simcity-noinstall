Detailed Description

Fake registry key handles.

To properly fool the game to not create registry entries and for the hooks to know when the game passes in fake keys, values starting with 0x80000040 correspond to fake keys that are never passed to the Win32 API. The Win32 API also uses predefined keys to address hives for example.
