Calculate the growth factor needed to fit videos in the game to the screen.

This function calculates and stores a growth factor in a global variable in the assembly implementation for use by the scale function.

Parameters
[in]widthWidth of the desktop monitor
[in]heightHeight of the desktop monitor

Desktop monitor is assumed to be the one where the game will be displayed. Multi-monitor setups have not been tested yet.
